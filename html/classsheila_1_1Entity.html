Detailed Description

template<class _Tp>
class sheila::Entity< _Tp >

Template class for an Entity in the SHEILa structure model.

Author
FlowerGenius
Template Parameters
_TpThe class that this Entity will represent.

An Entity is a special data type, it represents the pair of files required to build the class of the same name and it's variants.
