Detailed Description

The base for all of the specializations of CppClass.

Author
FlowerGenius

This struct provides a base object for all of the specializations of CppClass. This struct is not meant to ever be used in it's own right, rather it exists as a wrapper allowing any CppObject types to be accessed, stored, and used iteratively by upcasting to this base.
