Detailed Description

A Feeling as understood by SHEILa.

Author
FlowerGenius

To SHEILa, a Feeling is a specific map of Plutchik emotion values and their relations to one another. Since the number of unique feelings is not finite, an instance of the Feeling class represents a feeling, rather than having a class for each.
