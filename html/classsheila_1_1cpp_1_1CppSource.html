Detailed Description

An abstract model of a source file in C++.

Author
FlowerGenius

Each instance of this class models a C++ source file. This file can have a physical location on the disk, or exist entirely within memory. In either case this model can produce valid C++ code that can be written to a file, or valid XML code to be placed inside the definition of the object associated with this source file.
