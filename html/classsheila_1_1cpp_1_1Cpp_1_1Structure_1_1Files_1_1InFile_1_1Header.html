Detailed Description

An abstract model of a header file in C++.

Author
FlowerGenius

Each instance of this class models a C++ header file. A header file is either defined inside of object for the source file that it is first included from or has a virtual definition in the case of files outside of the project such as those in the C++ standard library and C++STL.
