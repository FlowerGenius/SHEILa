C++ Preprocessor directive specialization for a define directive.

An explicit specialization of the type DEFINE that represents an abstract model of a C++ preprocessor define directive.

idString that is a (typically) valid macro identifier.
statementString that is a (typically) valid macro expression.
argsVector of 0 or more strings for the macro arguments.

This constructor generates an abstract model of a C++ define directive. It also creates a new macro object representing the macro that has been defined by the directive.
